Function Documentation

state emergencyStop ( )

Elevator stops, queue and lights are reset and doors open for 3 seconds if the elevator is on a floor.

Returns
EM_STOP if STOP button is pressed down, or if there's less than 3 seconds since the last STOP button push. Else return IDLE
Warning
When IDLE is returned, lastDirection = DIRN_STOP

state moveOrWait ( )

Determines if the elevator state should switch to WAIT or MOVE. if the state is switched to MOVE, movement direction is also determined.

Returns
MOVE when there is a pending order. Return WAIT when the elevator has reached a floor with an active order in the same direction as the elevator.
Warning
can also return EM_STOP if some unforeseen error occurs.
