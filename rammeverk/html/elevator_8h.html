state determineDirectionOrWait ( )

Determines which direction the elevator needs to travel.

Returns
MOVE when the direction is either DIRN_UP or DIRN_DOWN, return WAIT when direction is DIRN_STOP.
Warning
can also return EM_STOP if some unforeseen error occurs.

state emergencyStop ( )

Stops the elevator for as long as STOP button is pressed, and an additional 3 seconds after release.

Returns
EM_STOP if STOP button is pressed down, or if there's less than 3 seconds since the last STOP button push. Else return IDLE
Warning
When IDLE is returned, lastDirection = DIRN_STOP
